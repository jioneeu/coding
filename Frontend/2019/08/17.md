## id와 class의 차이
- id
  * 단 하나의 요소를 특별하게 만들기 위해 사용. 중복 사용 (가능하나) 하면 안됨.
  ```css
  #some-id {
	property:value
  }
  ```
- class
  * 여러개의 요소들에 같은 효과를 줄 수 있다. 중복 사용 가능.
  ```css
  .some-class {
	property:value
  }
  ```
  * `id` overrides `class`
  * `class1` overrdise `class2`
  * Open in new tab: `target="_blank"`
