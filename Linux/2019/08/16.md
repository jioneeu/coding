### mkdir
mkdir을 사용해서 현재 위치에 새로운 디렉토리를 만들 수 있다.
```sh
$ mkdir 2019/

2019/
└── ./
└── ../
```

`2019/`폴더 안에 8월달에 공부한 내용을 담기위해 `08`폴더를 또 만들자.
```sh
$ cd 2019/
$ mkdir 08/

2019/
└── 08/
		└── ./
		└── ../
```

마지막으로 16일 내용을 위해 16이라는 폴더를 만들자.
```sh
$ cd 08/
$ mkdir 16

2019/
└── 08/
		└── 16/
				└── ./
				└── ../
```

## mkdir -p

위 과정을 `mkdir -p`를 이용해 한 줄로 표현할 수 있다.

```sh
mkdir -p /2019/08/16 && cd $_

2019/
└── 08/
		└── 16/
				└── ./
				└── ../
```
